# XIRH Suite

Site personnel de consultant et plateforme SIRH de démonstration.

## 🚀 Technologies
* **Framework** : Next.js (App Router, React 19)
* **Design** : Tailwind CSS v4, Glassmorphism, animations canvas interactives
* **Base de données** : PostgreSQL (géré via Prisma v6)
* **Hébergement & Déploiement** : Coolify / Docker (multi-stage build)

## 📁 Structure du projet
* `src/app/` : Routes Next.js (Landing page, Agenda, API REST)
* `src/views/` : Composants et pages de l'application SIRH
* `src/components/` : Composants UI réutilisables (Sidebar, Topbar, animations de fond)
* `prisma/` : Schéma de base de données PostgreSQL et scripts de peuplement (seeding)
